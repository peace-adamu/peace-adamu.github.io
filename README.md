# Adamu Peace - Portfolio Website

A modern, responsive portfolio website showcasing the work and achievements of Adamu Peace, an AI Engineer, Data Scientist, and Chemical Engineer.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**: Hover effects, smooth scrolling, and dynamic content
- **Professional Sections**: 
  - Hero section with animated elements
  - About section with education details
  - Professional experience timeline
  - Featured projects showcase
  - Awards and recognition
  - Contact form with email integration

## Technologies Used

- HTML5

## Quick Start

1. **Clone or Download** the files to your computer
2. **Open `index.html`** in your web browser to view the website locally

## Deployment Options

### Option 1: GitHub Pages (Recommended - Free)

1. **Create a GitHub account** if you don't have one
2. **Create a new repository** named `your-username.github.io` (replace with your actual GitHub username)
3. **Upload all files** (index.html, styles.css, script.js, README.md) to the repository
4. **Go to Settings** > **Pages** in your repository
5. **Select "Deploy from a branch"** and choose "main" branch
6. Your website will be available at `https://your-username.github.io`

### Option 2: Netlify (Free)

1. Go to [netlify.com](https://netlify.com)
2. Sign up for a free account
3. Drag and drop your project folder to deploy
4. Get a free subdomain or connect your custom domain

### Option 3: Vercel (Free)

1. Go to [vercel.com](https://vercel.com)
2. Sign up with GitHub
3. Import your repository
4. Deploy with one click

### Option 4: GitHub Codespaces

1. Upload files to a GitHub repository
2. Enable GitHub Pages in repository settings
3. Access your live website

## Customization

### Update Personal Information

1. **Contact Details**: Edit the contact section in `index.html`
2. **Projects**: Add or modify projects in the projects section
3. **Experience**: Update the timeline in the experience section
4. **Skills**: Modify the interests/skills section

### Add Your CV

Replace the `#` in the "Download My CV" button with a link to your actual CV file:

```html
<a href="path/to/your-cv.pdf" class="btn btn-outline" download>
```

### Customize Colors

The main color scheme uses CSS custom properties. You can change colors by modifying the gradient values in `styles.css`:

```css
/* Main brand colors */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Add More Sections

You can easily add new sections by following the existing pattern:

```html
<section class="new-section">
    <div class="container">
        <h2 class="section-title">New Section</h2>
        <!-- Your content here -->
    </div>
</section>
```
